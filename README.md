This Python script uses Natural Language Processing (Nutritionix API) to track workouts and logs them into a Google Sheet via the Sheety API.

## Setup
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`.
3. Set your Environment Variables for `APP_ID`, `API_KEY`, and `SHEETY_PASS`.
